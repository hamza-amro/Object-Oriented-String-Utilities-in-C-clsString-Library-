String Utilities Library in C++

A C++ project that implements a custom clsString class for advanced
string manipulation using Object-Oriented Programming (OOP) principles.

------------------------------------------------------------------------

FEATURES

-   Convert all letters to uppercase / lowercase
-   Convert first letter of each word
-   Invert letter cases (upper ↔ lower)
-   Count:
    -   Total letters
    -   Capital letters
    -   Small letters
    -   Specific letters
    -   Vowels
-   Split strings using custom delimiters
-   Count number of words in a string
-   Print first letter of each word

------------------------------------------------------------------------

-   C++
-   Object-Oriented Programming (OOP)
-   STL (vector, string)
